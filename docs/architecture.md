## 3. 工具系统架构

```
┌─────────────────────────────────────────────────────────────────────────────┐

## 4. LLM 客户端架构

```
┌─────────────────────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────────────────────┤

│  │  │  - model: str                                               │  │    │
│  │  │  - retry_config: RetryConfig                                │  │    │
│  │  │  - generate(messages, tools) -> LLMResponse                 │  │    │
│  │  └─────────────────────────────────────────────────────────────┘  │    │
│  │                              │                                       │    │
│  │         ┌────────────────────┼────────────────────┐               │    │
│  │         ▼                    ▼                    ▼               │    │
│  │  ┌─────────────┐    ┌─────────────┐    ┌─────────────────┐     │    │

│  │                     Base Interface: LLMClientBase                  │    │

## 6. 消息与数据流

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          Message Flow Schema                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  ┌─────────────────────────────────────────────────────────────────────┐    │
│  │                     Message Types                                   │    │
│  │                                                                     │    │
│  │  System:     Message(role="system", content="You are...")         │    │
│  │                                                                     │    │
│  │  User:       Message(role="user", content="Create a file...")    │    │
│  │                                                                     │    │
│  │  Assistant:  Message(                                              │    │
│  │                role="assistant",                                   │    │
│  │                content="I'll help you...",                         │    │
│  │                thinking="Let me think...",    (可选)              │    │
│  │                tool_calls=[...]              (可选)               │    │
│  │              )                                                      │    │
│  │                                                                     │    │
│  │  Tool:       Message(                                              │    │
│  │                role="tool",                                         │    │
│  │                content="File content...",                          │    │
│  │                tool_call_id="call_xxx",                           │    │
│  │                name="read_file"                                    │    │
│  │              )                                                      │    │
│  └─────────────────────────────────────────────────────────────────────┘    │
│                                                                              │
│  ┌─────────────────────────────────────────────────────────────────────┐    │
│  │                     Response Structure                             │    │
│  │                                                                     │    │
│  │  LLMResponse:                                                      │    │
│  │  {                                                                │    │
│  │    content: str,           // 文本响应                            │    │
│  │    thinking: str | None,   // 推理过程 (M2.5 extended thinking) │    │
│  │    tool_calls: [ToolCall], // 工具调用列表                        │    │
│  │    finish_reason: str,    // 结束原因 (stop/tool_use)            │    │
│  │    usage: TokenUsage      // Token 使用统计                       │    │
│  │  }                                                                │    │
│  └─────────────────────────────────────────────────────────────────────┘    │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```
