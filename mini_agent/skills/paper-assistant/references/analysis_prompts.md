# 论文分析提示词模板

本文档包含用于分析学术论文的提示词模板。根据不同的分析目标，选择合适的提示词。

---

## 1. 论文解析 - 元数据提取

### 提取论文基本信息
使用此提示词提取论文的标题、作者、摘要、关键词、DOI 等元数据。

```
请从以下论文内容中提取完整的元数据信息：

{论文内容}

请以 JSON 格式返回以下字段：
- title: 论文标题
- authors: 作者列表
- abstract: 摘要
- keywords: 关键词列表
- doi: DOI（如有）
- publication: 发表期刊/会议（如有）
- year: 发表年份（如有）
```

---

## 2. 内容理解 - 核心贡献总结

### 总结主要贡献
使用此提示词总结论文的核心贡献和创新点。

```
请分析以下论文的主要贡献：

标题：{论文标题}
摘要：{论文摘要}
结论：{结论部分}

请用简洁的语言回答：
1. 这篇论文试图解决什么问题？
2. 论文提出了什么新方法/新观点？
3. 该方法的核心创新点是什么？
4. 实验结果表明什么结论？
```

### 用通俗语言解释方法
使用此提示词将复杂的技术方法用通俗易懂的语言解释。

```
请用通俗易懂的语言解释这篇论文的研究方法：

{论文方法论部分}

要求：
1. 假设读者没有该领域的专业背景
2. 用日常生活中的例子类比
3. 解释核心概念和技术术语
4. 保持准确性同时追求可理解性
```

## 3. 结构分析 - 章节逻辑梳理

### 分析论文结构
使用此提示词分析论文的整体结构和逻辑流程。

```
请分析以下论文的整体结构和逻辑流程：

{论文全文或各章节标题}

请回答：
1. 论文分为哪些主要部分？
2. 各部分之间的逻辑关系是什么？
3. 论文的论证结构是怎样的？（如：总-分-总、递进式、对比式等）
4. 各部分的核心内容是什么？
```

### 绘制逻辑流程图
使用此提示词梳理论文的方法流程。

```
请梳理以下论文的方法论流程：

{方法论章节内容}

请用步骤化的方式描述整个研究过程：
步骤1: [做什么]
步骤2: [做什么]
...
最终: [得到什么结果]

同时指出每一步的关键技术和输入输出。
```

## 4. 笔记整理 - 结构化笔记生成

### 生成阅读笔记
使用此提示词生成结构化的论文阅读笔记。

```
请为以下论文生成一份结构化的阅读笔记：

{论文完整内容或关键部分}

笔记应包含：
1. 论文基本信息（标题、作者、年份、期刊）
2. 研究问题（研究背景、动机、目标）
3. 研究方法（数据、模型、实验设计）
4. 主要结论（核心发现、贡献、局限性）
5. 个人思考（与你研究的相关性、启发、应用场景）
```

### 生成思维导图
使用此提示词生成论文内容的思维导图结构。

```
请为以下论文生成思维导图结构：

{论文内容}

思维导图格式（Markdown）：
根节点: 论文标题
├── 第一层级节点1
│   ├── 第二层级节点
│   └── 第二层级节点
├── 第一层级节点2
│   ├── 第二层级节点
│   └── 第二层级节点
└── ...

请覆盖：背景、方法、实验、结论、相关工作 等主要方面。
```

## 5. 术语解释 - 专业概念解析

### 解释专业术语
使用此提示词解释论文中的专业术语和概念。

```
请解释以下论文中出现的专业术语：

{术语列表或包含术语的上下文}

对于每个术语，请提供：
1. 术语名称
2. 简要定义（1-2句话）
3. 在本文中的具体含义
4. 如果有通俗类比，也可以提供
```

---

## 6. 时间线 - 论文历史梳理

### 梳理论文历史
使用此提示词梳理论文的相关历史和引用关系。

```
请根据以下信息梳理论文的历史脉络：

论文标题：{标题}
DOI：{DOI}
关键词：{关键词}

请回答：
1. 这篇论文的最早相关研究是什么时候开始的？
2. 该领域的关键里程碑有哪些？
3. 这篇论文站在哪些前人研究的基础上？
4. 该领域的后续研究是如何发展的？
```

## 7. 问答交互 - 深度理解

### 针对内容提问
使用此提示词根据论文内容回答具体问题。

```
基于以下论文内容：

{论文内容}

请回答用户的问题：

问题：{用户问题}

要求：
1. 直接基于论文内容回答
2. 如果论文没有涉及该问题，请明确说明
3. 如需补充外部信息，请说明来源
```

### 深入讨论
使用此提示词就论文某个方面进行深入讨论。

```
请深入讨论以下论文的某个方面：

论文：{论文标题}
方面：{讨论方面，如：方法论、实验设计、创新点、局限性等}

请从以下角度进行分析：
1. 这个方面具体是什么？
2. 论文是如何处理这个方面的？
3. 这个方面的处理方式有什么优点？
4. 有没有可以改进的地方？
5. 对后续研究有什么启示？
```

## 使用提示

1. **选择合适的模板**：根据你的分析目标选择对应的提示词
2. **填充占位符**：将 `{...}` 中的内容替换为实际的论文内容
3. **调整提示词**：根据具体需求调整提示词的细节和格式
4. **结合使用**：可以组合多个提示词进行全面分析
5. **迭代深化**：初步分析后，可以针对特定问题进一步提问
